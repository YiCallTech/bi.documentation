# Schema

```
"structure":{}
```
structure中参数具体含义详见 - https://echarts.apache.org/zh/option.html#radar

### 需要动态赋值的数据如下所示
#### schema.radar.radius
| 名称 | 描述 | 必填 | 类型 |备注 |
|--|--|--|--|--|
| radius| 半径 | 否 | number/string |  |

#### schema.option.series[0].data
| 名称 | 描述 | 必填 | 类型 |备注 |
|--|--|--|--|--|
| name | 名称 | 是 | string | 各项指标名称 |
| value | 值 | 是 | array | 各项指标值 |
