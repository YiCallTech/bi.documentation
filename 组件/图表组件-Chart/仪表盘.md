# Schema

```
"structure":{}
```
structure中参数具体含义详见 - https://echarts.apache.org/zh/option.html#series-gauge

### 需要动态赋值的数据如下所示
#### schema.option.series.data
| 名称 | 描述 | 必填 | 类型 |备注 |
|--|--|--|--|--|
| name | 名称 | 是 | string | 各项指标名称 |
| value | 值 | 是 | int | 各项指标值 |
